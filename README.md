# nginx-deploy

nginx 项目部署 WEB 服务

nginx 的简单命令

- nginx -s stop 快速停止服务

- nginx -s reload 配置文件改动，需重新载入

- nginx -s quit 正常停止服务

- start nginx 启动

值得注意的是在 window 系统下，适当增加 .\nginx 来启动，否则不识别此命令

例如 .\nginx -s stop
